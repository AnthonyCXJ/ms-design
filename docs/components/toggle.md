### API

#### Toggle props
| 属性 | 说明 | 类型 | 可选值 | 默认值 |
| :------------ | :------------ | :------------ | :------------ | :------------ |
| checked | 是否被选中, 可以使用 `v-model` 双向绑定数据 | Boolean | `true` \| `false` | `false` |
| disabled | 是否禁用 | Boolean | `true` \| `false` | `false` |

#### Toggle events
| 事件名 | 说明 | 返回值 |
| :------------ | :------------ | :------------ |
| change | 开关被点击选项状态发生变化时触发 | true \| false |
