### API

#### Toggle props
| 属性 | 说明 | 类型 | 默认值 |
| :------------ | :------------ | :------------ | :------------ |
| checked | 是否被选中, 可以使用 `v-model` 双向绑定数据 | Boolean | false |
| disabled | 是否禁用 | Boolean | false

#### Toggle events
| 事件名 | 说明 | 返回值 |
| :------------ | :------------ | :------------ |
| change | 被点击时选项状态发生变化 | true \| false |
